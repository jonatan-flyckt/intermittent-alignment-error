<h1>Intermittent Alignment Error</h1>

<p><strong>Timing-aware metric for intermittent demand forecasting</strong></p>

<p>
This branch preserves the version of the <em>Intermittent Alignment Error</em> metric used in the licentiate thesis:
</p>

<p>
<a href="https://urn.kb.se/resolve?urn=urn:nbn:se:bth-28531"><em>Decision Support through Global Demand Forecasting</em></a>
</p>

<p>
The metric is designed to evaluate point forecasts for intermittent demand while taking both timing and magnitude into account.
</p>

<p>
The current and maintained version of the implementation is available in the <code>main</code> branch.
</p>
